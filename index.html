My Projects

In this project,the objective was to use a mockup image also known as comp, source text and images,

This project teaches use to use a mockup image to learn how to use
Header,Paragraph,Image and Span Tags to make a beautiful Album Cover.
It is very applicable in teaching HTML basics and structuring of both text and images.
